HTML Structure and Tags

Learn about the fundamental structure of an HTML document and its basic tags.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Main Heading</h1>
    <p>Paragraph content...</p>
    <h2>Sub-Heading</h2>
    <p>More paragraph content...</p>
  </body>
</html>

Basic HTML Tags

  • <head> - Contains meta information about the HTML page
  • <title> - Specifies a title for the HTML page
  • <body> - Defines the document's body, contains all visible contents
  • <h1> - Defines the most important heading
  • <p> - Defines a paragraph
